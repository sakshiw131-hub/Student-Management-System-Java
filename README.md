🎓 Student Management System

📌 Project Description

The Student Management System is a console-based application developed using Core Java. This project is designed to manage student records efficiently and demonstrates the implementation of Object-Oriented Programming (OOP) concepts.

✨ Features

- Add Student Details
- View All Student Records
- Search Student by ID
- Update Student Information
- Delete Student Records
- Menu-Driven Console Application

🛠️ Technologies Used

- Core Java
- Object-Oriented Programming (OOP)
- Eclipse IDE
- Git & GitHub

📁 Project Structure

- "src" - Contains all Java source files
- "Screenshot" - Contains output screenshots of the project

▶️ How to Run

1. Open the project in Eclipse IDE.
2. Run the "Main.java" file.
3. Follow the menu options displayed in the console.

- Java Programming Fundamentals

- Array/List Operations
- Console-Based Application Development
